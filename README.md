# Medico-webcare

Medico web care is an online consultancy service website which helps patient and doctor to communicate easily in a virtual mode at any time.
It’s a web based service which deals with data of patient and doctor and provide an easily way to connect with each other.

It contains as a patient view which will access login portion to identify particular identity of a patient, patient will have all part
of information at their fingertip such as patient profile , checkup details , report details  , payment transaction info and patient can
also view the doctor details and select as per his/her choice for the consultancy services.

Now, it also has a doctor view which will access login portion to identify particular identity of a doctor and it also have same part 
of information as patient view as doctor profile, patient info which will contain sub all info of patient as report and data present
based on particular patient.
 
Part of consultancy services it contains two side webpage as a patient and  a doctor , the medium of communication is a message box
which is responsive on both side to connect in a easily manner . Co-relation of data in connected to both side for the flow of
message box and doctor can view all important info of patient on a fingertip and patient can also view some info of doctor this
lead to better way of related data of a particular identity.

The medico webcare is essentially an website which focuses on overall consultancy services, it will contains all data of patient checkup details and info

It is a full stack – web development project.

The code is written on HTML,CSS, JAVASCRIPT ,NODEJS ,EXPRESSJS ,MONGODB.
